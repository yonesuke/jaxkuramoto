# jaxkuramoto
![PyPI - Python Version](https://img.shields.io/pypi/pyversions/jaxkuramoto)

JAX implementation of the Kuramoto model.


## Installation
`jaxkuramoto` can be installed with pip with the following command:
```
pip install jaxkuramoto
```
